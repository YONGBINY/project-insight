🧠 Project Insight: 인지 프로파일링 챌린지
=
"똑같이 7문제를 맞혔다면, 두 학생은 같은 수준일까?"

###### 이 프로젝트는 '정답 개수'라는 1차원적 결과 뒤에 숨겨진, 다차원적인 **'문제 해결 과정'**의 차이를 데이터로 증명하여, 획일화된 교육의 한계를 넘어선 초개인화 교육의 필요성을 제시하고자 시작되었습니다.  
###### ![alt text](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)  
###### (↑ 클릭하여 챌린지에 직접 참여해보세요!)

## 1. 프로젝트 배경 및 문제 인식  
- 시작 의문: 전통적인 평가는 '결과'에만 집중한다. 하지만 같은 결과를 냈더라도, 문제를 푸는 속도, 전략, 어려움을 느끼는 지점 등 '과정'은 천차만별이다. 즉, 이런 획일화된 공교육 시스템은 개인의 잠재력을 온전히 이끌어내지 못하고 교육 격차를 심화시킬 것이다. 
- 문제 인식: 이 '과정의 데이터'를 측정하고 분석할 수 있다면, 각 학습자의 고유한 강점과 약점을 파악하여 최적화된 학습 경험을 제공할 수 있다. 이는 곧, **'AI 기반 초개인화 교육'**의 핵심적인 첫걸음이 될 수 있을 것이다.
## 2. 핵심 목표
데이터를 통해 학습자의 숨겨진 **인지 패턴(Cognitive Patterns)**를 가시화하고, 이를 바탕으로 다양한 **학습자 유형(Persona)**을 정의한다.

## 3. 솔루션 설계: 어떻게 풀어나갈 것인가?
사용자의 '생각의 흐름'을 포착하기 위해, 다음과 같이 솔루션을 설계하고 필요한 도구를 정의했습니다.
| 단계              | 목표                      | 핵심 도구 및 방법론                                                  |
| --------------- | ----------------------- | ------------------------------------------------------------ |
| **1\. 데이터 수집**  | 사용자의 문제 해결 과정을 실시간으로 기록 | 🧠 **인지 프로파일링 챌린지** (웹 앱)<br> Streamlit으로 빠른 프로토타이핑          |
| **2\. 데이터 저장**  | 수집된 로그를 영구적이고 안정적으로 관리  | ☁️ **클라우드 기반 데이터베이스**<br> gspread를 활용, Google Sheets에 실시간 적재 |
| **3\. 즉각적 피드백** | 사용자의 참여 유도 및 바이럴 확산     | 📊 **규칙 기반 실시간 페르소나 분석**<br> Pillow를 활용한 결과 이미지 공유 기능        |

### [근거] 왜 이 문제들을 채택했는가?  
단순한 지식 테스트가 아닌, '사고 과정' 자체를 측정하기 위해 **'인지 과정 가시성(Cognitive Process Visibility)'**이 높은 문제들을 전략적으로 선정했습니다.  
- 수학/논리 사고: 구조적, 논리적 사고의 흐름 측정.
- 과학적 추론: 데이터 기반의 가설 설정 및 검증 능력 측정.
- 컴퓨팅 사고력: 복잡한 문제를 단순한 절차로 분해하는 능력 측정.

## 4. 구현 과정 (Tech Stack)
- Frontend & Prototyping: `Streamlit`
- Data Logging & Storage: `gspread`, `Google Sheets API`
- Data Analysis (예정): `Pandas`, `Scikit-learn`
- Image Generation: `Pillow(PIL)`
- Deployment & CI/CD: `GitHub`, `Streamlit Community Cloud`

## 5. 향후 계획 (Roadmap)
Project Insight는 이제 시작입니다. 수집된 데이터를 통해 더욱 깊이 있는 분석과 가치 창출을 계획하고 있습니다.

- Phase 1: 로그 데이터 심층 분석 (진행중)  
  - Jupyter Notebook을 활용하여 수집된 원본 로그 데이터에 대한 탐색적 데이터 분석(EDA) 수행.
  - 시간, 정답률, 힌트 사용 패턴 등 핵심 지표 간의 상관관계 분석 및 시각화.
- Phase 2: 머신러닝 기반 페르소나 고도화
  - K-Means 클러스터링 등 비지도 학습 알고리즘을 적용하여, 데이터 기반의 객관적인 학습자 유형(페르소나) 발견.
  - 현재의 '규칙 기반' 모델을 '데이터 기반' 모델로 업데이트하여 분석의 정확도와 설득력 강화.
- Phase 3: 공공 데이터 연계 분석
  - 사용자 정보(연령, 학력 등)와 국가 통계 데이터(e.g., 지역별 학업 성취도, 소득 수준)를 연계 분석.
  - 사회경제적 배경이 인지 패턴에 미치는 영향 등 거시적인 교육적 인사이트 발굴 시도.
